# vectorTectonic

`vectorTectonic` ist ein geplantes, deterministisches System zur prozeduralen Simulation plausibler Plattentektonik auf einer Kugeloberfläche.

Der Simulationszustand besteht aus vektorbasierten Punkten, Linien und sphärischen Polygonen – nicht aus einem Raster. Rust bildet den Simulationskern; Bevy visualisiert den Ablauf und die finale Feature-Map als interaktiven 3D-Globus.

Die fachliche und technische Spezifikation wird zunächst über eine Wayfinder-Map im Issue-Tracker erarbeitet. Eine Heightmap, Geländeverfeinerung und GPlates-Kompatibilität gehören nicht zum ersten Vorhaben.

## Status

Planungsphase. Die Entscheidungen werden in der Wayfinder-Map [Polygonbasierte Tektonik-Simulation spezifizieren](https://github.com/timjonaswechler/vectorTectonic/issues/1) erarbeitet.
